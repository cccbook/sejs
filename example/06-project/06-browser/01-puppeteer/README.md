# 練習 e2e 測試 -- 使用 puppeteer

* Puppeteer : Chrome headless browser
* e2e : end to end test

1. 安裝 puppeteer
  * npm i puppeteer
2. 撰寫 puppeteer 測試程式 example.js
  * 參考 [example.js](example.js)
3. 執行 puppeteer 範例
  * node example.js
4. 觀看 [example.png](example.png) 截圖畫面
